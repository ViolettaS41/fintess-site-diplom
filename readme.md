# Сайт для фитнес-центра с записью на тренировки _дипломный проект_

Веб-сайт, позволяющий клиентам ознакомиться с фитнес-центром, зарегистрироваться на сайте, просматривать тренировки и записываться на них, а администраторам - совершать CRUD операции над данными тренеров, клиентов и составлять расписание тренировок.

### Технологии
`React`
`FastAPI`
`MySQL`
`Vite`
`JWT`

### Возможности системы
🪪 **Для клиентов:** 
- Регистрация и авторизация
- Просмотр расписания тренировок
- Запись на тренировку
- Отмена и подтверждение записи
- Просмотр личного кабинета

👩‍💻 **Для администратора:** 
- Управление пользователями (CRUD)
- Управление тренерами (CRUD)
- Управление расписанием тренировок
- Подтверждение и отмена записей

## Инструкция как запустить и установить приложение

1. Скачать репозиторий из GitHub

2. Открыть терминал
##### **Запуск фронтенд**:
3. Перейти в папку *frontend* -> *react-frontend*

4. Прописать команду ***npm install*** для установки зависимостей

5. Прописать команду ***npm run dev*** для запуска фронтенд-части
##### **Запуск бекенд**:
6. Открыть еще один терминал

7. Активировать venv, если он все еще не активирован (*python -m venv venv* -> *venv\Scripts\activate*)

8. Перейти в папку *backend* 

9. Установить зависимости ***pip install -r requirements.txt***

10. Прописать команду ***uvicorn main:app --reload*** для запуска бэкенд-части
